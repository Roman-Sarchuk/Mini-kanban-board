# Mini Kanban Board 

A minimal Kanban board built with React.

This project demonstrates state management, custom hooks, drag-and-drop interactions, and persistent storage using localStorage.


> # 🚀 Features
> * 3 columns: To Do, In Progress, Done
> * CUID operations (Create, Update, Delete) implemented via a custom hook
> * Drag & Drop powered by @dnd-kit
> * Inline editing of tasks (via right-click interaction)
> * Data persistence using localStorage


> # 🛠 Tech Stack
> * Node.js
> * Vite
> * React
> * @dnd-kit
> * Custom React hooks
> * localStorage API


> # 🧠 What This Project Demonstrates
> * Managing complex state in React
> * Creating reusable custom hooks
> * Implementing drag-and-drop logic
> * Handling inline editing UX
> * Persisting application state in the browser
> * Clean component structure and separation of concerns
